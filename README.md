# hippocamp

This repository documents the progress in the project to create a small-scale makercamp in the netherlands, in the style of bornhack.dk

Current state: exploration.

Targets:
- small-scale, max 500 people
- community oriented, village-led
- not a festival
- tech is important, but not the focus
- yearly cadence

The idea is to start small, and try to build a culture that allows us to get away with very few written rules.
